## [1.0.0] - 2025-01-01

### Added
- 📊 Real-time system dashboard (CPU, RAM, disk, GPU temp, throttle)
- 🖥️ Browser-based terminal with full PTY support
- 📁 File manager with upload, download, rename, delete
- ⚙️ systemd service manager with live log streaming
- 📦 apt package update manager
- 🌐 LAN network scanner with ping
- 🔌 GPIO pin monitor (Raspberry Pi 5 native via lgpio)
- 🌗 Dark and light theme with toggle
- 🔒 Password authentication with token sessions
- 📟 `pipanel` CLI for start/stop/restart/update/uninstall
- One-line `curl | bash` installer with systemd service setup
